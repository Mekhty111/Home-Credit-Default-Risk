# data/processed/

This directory stores intermediate processed files generated by the notebooks.

## Contents

| File | Generated by | Description |
|---|---|---|
| `master_table.parquet` | `03_feature_engineering.ipynb` | Master table with all 7 tables joined |
| `train_woe.parquet` | `04_woe_iv_selection.ipynb` | Train set with WoE-transformed features |
| `oot_woe.parquet` | `04_woe_iv_selection.ipynb` | OOT set with WoE-transformed features |

## Note

These files are excluded from version control (`.gitignore`).  
To regenerate, run the notebooks in order:
```
03_feature_engineering.ipynb → 04_woe_iv_selection.ipynb
```
